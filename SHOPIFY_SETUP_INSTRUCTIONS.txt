üì¶ STEP 1: INSTALL DEPENDENCIES
---------------------------------
Before starting the server, install axios:

  cd backend
  npm install axios

Or if using yarn:

  cd backend
  yarn add axios


üîß STEP 2: CONFIGURE SHOPIFY APP
---------------------------------
1. Go to Shopify Admin ‚Üí Settings ‚Üí Apps and sales channels ‚Üí Develop apps
2. Create new app: "Buysial Dropshipping"
3. Configure Admin API scopes:
   ‚úì read_products
   ‚úì write_products
   ‚úì read_orders
   ‚úì write_orders
   ‚úì write_fulfillments
   ‚úì read_inventory
   ‚úì write_inventory
4. Install app and copy Admin API access token (shpat_...)


‚öôÔ∏è STEP 3: CONFIGURE BUYSIAL
---------------------------------
1. Start your BuySial server
2. Login to web.buysial.com
3. Navigate to: User ‚Üí Shopify Integration
4. Enter:
   - Shopify Store URL: your-store.myshopify.com
   - Admin API Access Token: shpat_...
   - API Version: 2024-01
   - Webhook Secret: (optional)
5. Click "Save Shopify Settings"


ü™ù STEP 4: SET UP WEBHOOKS
---------------------------------
In Shopify Admin ‚Üí Settings ‚Üí Notifications ‚Üí Webhooks:

Create 3 webhooks:

1. Order creation
   URL: https://web.buysial.com/api/shopify/webhooks/orders/create
   Format: JSON

2. Order fulfillment
   URL: https://web.buysial.com/api/shopify/webhooks/orders/fulfilled
   Format: JSON

3. Order cancellation
   URL: https://web.buysial.com/api/shopify/webhooks/orders/cancelled
   Format: JSON


üì¶ STEP 5: SYNC PRODUCTS
---------------------------------
1. Go to: Products ‚Üí In-house Products
2. Create or edit products
3. Check ‚òë "Sync to Shopify" checkbox
4. Save product
5. Go to: User ‚Üí Shopify Integration
6. Click "Sync All Products"
7. Products will appear on your Shopify store!


‚úÖ STEP 6: TEST THE INTEGRATION
---------------------------------
1. Place a test order on your Shopify store
2. Check BuySial ‚Üí Orders ‚Üí Orders List
3. You should see the order with "Shopify" source
4. Process the order normally
5. Mark as shipped ‚Üí Shopify will be updated automatically!
